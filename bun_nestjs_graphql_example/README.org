To install dependencies:

#+begin_src sh
bun install
#+end_src

To build the React application:

#+begin_src sh
bun run build:react
#+end_src

To initialize the database with sample data:

#+begin_src sh
bun run init:db
#+end_src

To run the application:

#+begin_src sh
bun run start:dev
#+end_src

To clean the database and reinitialize:

#+begin_src sh
bun run clean:db
#+end_src

You can access:
- GraphQL Playground: http://localhost:3010/graphql
- React UI Example: http://localhost:3010/

This project was created using =bun init= in bun v1.1.42.
[[https://bun.sh][Bun]] is a fast all-in-one JavaScript runtime.
